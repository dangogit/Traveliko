# Traveliko

Mobile travel planning app with country exploration, trip building, and an AI chat assistant.

## Tech Stack

- React Native, Expo SDK 52
- Expo Router (file-based navigation)
- TypeScript
- Zustand (state management)
- NativeWind (Tailwind for React Native)
- Lucide icons

## Features

- Browse countries, regions, and attractions with detailed info cards
- Multi-step trip creation wizard (countries, start/end points, preferences, summary)
- Day-by-day trip itinerary builder with activity management
- Accommodation and transportation search
- Weather forecasts with activity recommendations
- AI chatbot for travel suggestions
- Favorites and profile setup with interest/budget selectors
- Onboarding flow

## Screens

38+ screens including trip planner, country/region/location details, weather, recommendations, search, and chat.

## Getting Started

```bash
git clone https://github.com/dangogit/Traveliko.git
cd Traveliko
npm install
npx expo start
```
